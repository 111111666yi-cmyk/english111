# v1.7 上架说明与更新文案

## App Review Notes (EN)

This release focuses on content quality, audio resource governance, and dual-platform stability.

We introduced a release-grade content audit pipeline to improve the compliance and accuracy of the learning library. The formal study flow now uses 1,470 reviewed words, while additional searchable reference entries remain available in the library with a clear "library-only" label and do not enter tests, review, challenge, or progress metrics.

Core learning content and packaged audio are bundled with the app and remain available offline after installation. `INTERNET` is retained for standard webview/network compatibility and optional user-triggered online features; no storage or media permission is requested because the app does not scan or write shared device files in the background.

We also optimized offline audio management by validating packaged references, checking for orphaned resources, and verifying that Android packaged assets stay in sync with the exported web build. We have verified the integrity of all packaged learning audio references, including the 1,470 reviewed study words and their related sentence, passage, expression, and UI audio. This improves playback stability and keeps resource usage cleaner.

In addition, we completed a full dual-platform verification pass for Web and Android. No new sensitive Android permissions were added in this release; the app still only declares `INTERNET`.

## 审核说明（中文参考）

本次更新重点优化了应用内的音频资源管理，清理了冗余的离线资源映射，并提升了词库内容的合规性与准确性。我们新增了发布级内容审计机制，将正式学习内容与仅词库保留内容分层；当前正式学习流仅使用 1470 个已审查词汇，其余词条仍可在词库中检索，但会明确标记为“仅词库保留”，不会进入测试、复习、闯关和进度统计。

核心学习内容与打包音频会随安装包一并提供，安装后即可离线使用。当前保留 `INTERNET` 权限是为了 WebView/网络兼容和用户主动触发的在线功能；应用不会在后台扫描或写入共享存储，因此未申请存储或媒体权限。

同时，我们补强了 Web 与 Android 双端的构建、资源同步和回归校验流程，已完成全部打包学习音频引用完整性核验，覆盖正式学习词、句子、短文、表达与界面音频资源，提升了在最新系统环境下的运行稳定性。本次更新未新增 Android 敏感权限，当前仍只声明 `INTERNET`。

## What's New

- 【内容精修】深度清洗了核心词库，学习内容更精准，告别重复和错误。
- 【音频优化】全面治理音频资源，听力练习更流畅，占用空间更精简。
- 【体验升级】修复了多项已知问题，提升了在双端设备上的运行稳定性。
