# v1.7 上架说明与更新文案

## App Review Notes (EN)

This release focuses on content quality, audio resource governance, and dual-platform stability.

We introduced a release-grade content audit pipeline to improve the compliance and accuracy of the learning library. Release-ready learning materials are now separated from library-only reference content, so the main study flow uses only reviewed content.

We also optimized offline audio management by validating packaged references, checking for orphaned resources, and verifying that Android packaged assets stay in sync with the exported web build. This improves playback stability and keeps resource usage cleaner.

In addition, we completed a full dual-platform verification pass for Web and Android. No new sensitive Android permissions were added in this release; the app still only declares `INTERNET`.

## 审核说明（中文参考）

本次更新重点优化了应用内的音频资源管理，清理了冗余的离线资源映射，并提升了词库内容的合规性与准确性。我们新增了发布级内容审计机制，将正式学习内容与仅词库保留内容分层，确保主学习链路只使用通过审查的高质量学习素材。

同时，我们补强了 Web 与 Android 双端的构建、资源同步和回归校验流程，提升了在最新系统环境下的运行稳定性。本次更新未新增 Android 敏感权限，当前仍只声明 `INTERNET`。

## What's New

- 【内容精修】深度清洗了核心词库，学习内容更精准，告别重复和错误。
- 【音频优化】全面治理音频资源，听力练习更流畅，占用空间更精简。
- 【体验升级】修复了多项已知问题，提升了在双端设备上的运行稳定性。
